# MistOS
MistOS 是一个基于 Scratch 开发的 WebOS **交互程序**。它本质上是一个在 TurboWarp 平台上运行的、模仿操作系统界面的程序，**并非**运行在真实计算机硬件（如 x86_64 架构）上的操作系统。MistOS 完全使用 Scratch 积木搭建，依赖 Scratch 引擎运行。<br>
**如果你想体验或创作ScratchOS，欢迎试试 MistOS！**


# 库结构
|目录|描述|备注|
|:-:|:-:|:-:|
|**Online**|MistOS 在线资源获取目录|可以从这里下载MistOS的内置资源|
|**Dev**|开发版本（功能最新，稳定性较低）|不定期更新|
|**Snapshot**|预览版本（功能较新，用于测试）|更新较快|
|**Release**|正式稳定版本|更新最慢|
|**HoaxVersion**|愚人节特别版本（仅恶搞）|每年四月左右更新|
